# 📚 IPAS 營運智慧分析師 學習筆記
## 【單元 C】經營管理與專案規劃 (PEST/波特五力/BCG/BPR流程/專案限制)

本單元整合了考科二「經營管理基本知識」與「數位化轉型創新與價值創造」以及考科一「評估與規劃營運智慧」中有關**企業機能**、**策略管理**、**流程再造 (BPR)** 與 **專案管理 (Gantt/PERT/CPM)** 的所有必考重點。

---

### 💡 核心生活譬喻：如何規劃與管理一間會賺錢的「連鎖手搖飲店」？

*   **策略管理**：手搖飲店老闆決定「品牌定位」（發現健康意識抬頭，且自家有有機茶葉契作，決定主打「高質感低糖有機茶」）。
*   **價值鏈中的主要活動**（第一線接觸飲料與客人的活動）：研發新茶配方 ➔ 向茶農進貨 ➔ 店員搖飲料（生產） ➔ 外送員送飲料（配送） ➔ 買一送一廣告（銷售） ➔ 處理飲料做錯的客訴（售後）。
*   **價值鏈中的支援活動**（後勤支柱）：採購部向茶農議價、人資部招募員工、IT 維護點餐 POS 系統。
*   **BPR (企業流程再造)**：原本排隊點單，櫃檯手寫單子常做錯又慢。老闆徹底推翻舊流程，引進貼紙機與電子看板，出餐快 3 倍。這就是徹底翻新的 **BPR**。
*   **專案三重限制**：想要飲料店新裝修（專案），會受到 **「預算（成本）」**、**「開幕時間（時間）」** 與 **「裝潢設計（範疇）」** 的限制。其中一項改變，其他都會受影響。

### 1. 企業經營環境與策略管理

#### 錢德勒 (Chandler) 聯結理論：
> [!NOTE]
> 錢德勒指出：**「環境 ➔ 策略 ➔ 結構」**。企業必須隨環境的變動調整經營策略，進而調整組織結構以維持競爭力。

#### 🏢 內部環境 vs 外部環境
*   **內部環境**：屬於組織之內部因素。如股東、董事會、組織文化、組織結構、員工態度、管理程序。
*   **外部環境**：分為特定環境與一般環境：
    *   **特定環境 (Specific Environment)**：對組織有直接影響的外部力量。包括：**顧客群、供應商、競爭群、金融機構、股東群、政府、壓力團體**。
    *   **一般環境 (General Environment / PEST-EL)**：間接影響企業的外部力量。包括：**經濟、政治與法律、社會文化、科技、人口結構、自然生態、國際化**。
        *   *出題盲點*：居民的風俗習慣、宗教信仰、文化水平，歸類在一般環境中的 **「社會文化環境」**。

#### 🛠️ 策略分析工具
1.  **波特五力分析 (五種競爭動力)**：評估特定環境的工具。包括：供應商的談判能力、顧客的議價能力、潛在進入者的威脅、替代品的威脅、產業現有廠商間的競合（加入「其他關係人」即為**六力分析**）。
2.  **BCG 矩陣 (事業組合分析)**：以市場成長率與相對市占率，評估資源分配：
    *   *笨狗 (Dogs)*：低市占、低成長。
    *   *問題 (Question Marks)*：低市占、高成長。
    *   *明星 (Stars)*：高市占、高成長。
    *   *金牛 (Cash Cows)*：**處於成熟期，高市占、低成長**，能為公司創造穩定的現金流。
3.  **波特價值鏈模型 (Value Chain)**：
    *   **主要活動 (Primary)**：進貨物流、生產作業、出貨物流、行銷與銷售、售後服務。
    *   **支援活動 (Support)**：採購、人力資源管理、技術研發、企業基礎設施。
    *   *用途*：價值鏈模型提供 **「檢驗基準 (Benchmark)」**，讓企業組織與同行進行執行效率比較。

### 2. 企業核心機能流程與管理活動 (產、銷、人、發、財)

#### 🏭 A. 生產與作業管理 (製造業 vs 服務業)
*   **製造業**：結果導向、產品具體、品質可客觀量測、顧客連結度低。
*   **服務業特色**：程序導向、主觀品質認定、顧客連結度高。四大特色：
    *   *無形性 (Intangibility)*：無法觸摸，但顧客能體驗或感受服務品質。
    *   *不可分割性 (Inseparability)*：服務的**提供與消費是同時且互動完成**的。
    *   *易變性 (Variability)*：服務人員與情境差異，造成服務品質易變。
    *   *易消逝性 (Perishability)*：服務無法儲存，過期即消逝。
*   **即時化生產 (JIT, Just in time)**：關鍵在於製造廠商與供應廠商必須培養 **「合夥關係」**。

#### 📣 B. 行銷管理 (Marketing)
*   **行銷組合 (4Ps)**：產品 (Product)、價格 (Price)、通路 (Place)、推廣 (Promotion)。
*   **產品生命週期 (PLC)**：
    *   *導入期*：銷售量難以起色，推銷費用高，利潤為負，產品設計應儘量簡單。
    *   *成長期*：**銷售量急遽攀升**，產品被市場接受，利潤大幅增加。
    *   *成熟期*：銷量與利潤達到高峰，競爭最激烈，適合使用 **「競爭導向」定價**。
    *   *衰退期*：銷量下降，應淘汰不賺錢產品。

#### 🔬 C. 研究與發展 (R&D)
*   **基礎研究 (Basic)**：目的在於求知以發現知識，增加人類知識，屬抽象概念。
*   **應用研究 (Applied)**：將基礎研究知識運用於解決實務問題，改善人類狀況。
*   **技術特性與介面管理**：
    > [!WARNING]
    > 開發與應用階段的重疊區域稱為 **介面 (Interface)**。介面管理不當易導致 **技術片段性 (Fragmentation)** 問題，無法達成一致性。
*   **R&D 兩大特質**：**非例行性活動**、**高度風險性**。

#### 👥 D. 人力資源管理 (HRM)
*   **工作說明書**：書面說明員工執行特定工作必備的基本資格、知識、技能與能力。
*   **工作擴張三類型**（依事件順序排列）：
    1.  **工作擴大化**（橫向擴大，如增加同水平的在職訓練工作）。
    2.  **工作豐富化**（縱向擴大，如增加規劃與考評工作權限）。
    3.  **工作輪調**（職位間輪換，如從教育訓練調至招募崗位）。
*   *招募方式*：內部招募（晉升調職，成本低但缺乏新創意）與外部招募（吸引具創意人才，適合創新與擴張策略）。

#### 💰 E. 會計與財務管理
*   **會計**：著重於**歷史資訊**的整理、記錄與報告（過往的交易數據）。
*   **財務管理**：著重於**未來的財務資金規劃**、預測與金融決策。
*   > [!IMPORTANT]
    > **稽核人員 (Auditor)**：執行企業內部控制，主要針對生、銷、會、研、財作業流程進行稽查，地位必須 **獨立向董事會報告**。
*   **資產負債表**：說明企業在 **「某一特定日期」** 的財務狀況。
*   **流動比率**：衡量企業的 **「清償能力（短期償債能力）」**。

### 3. 企業流程再造 (BPR)

*   *定義*：對企業運作模式進行根本性重新思考、徹底翻新作業流程。
*   *四大重點*：**根本 (Fundamental)**、**徹底 (Radical)**、**大幅度 (Dramatic)**、**流程 (Process)**。
*   **Davenport and Short 五步驟模型**（順序必考，簡記為 **52143**）：
    > [!IMPORTANT]
    > 1. **建立企業願景及活動目的** ➔ 2. **確定需要重新設計的流程** ➔ 3. **瞭解現行流程** ➔ 4. **確定資訊科技的功用** ➔ 5. **建立新流程的雛型**

### 4. 評估與規劃營運智慧（專案管理工具）

#### 📊 甘特圖 vs PERT / CPM
*   **甘特圖 (Gantt Chart)**：低成本，能監控簡單專案中各活動進度、先後順序與工期。**缺點：無法描繪活動與資源間的相互關係**。
*   **計畫評核技術 (PERT) & 要徑法 (CPM)**：可確實考慮活動間的先行關係與相依關係。
    *   *CPM (要徑法)*：藉由關鍵路徑（最長時程路徑）分析，決定計畫**最短完成時間**。
    *   *PERT (計畫評核技術)*：使用 **樂觀時間、悲觀時間、最可能時間** 等三個預期值，計算特定任務預期時間。

#### ⚠️ 專案限制與範圍蔓延
*   **專案三重限制 (Triple Constraints)**：**成本、時間、範疇 (Scope)**。任何一個因素改變，其他兩個因素都會受到連帶影響。
*   **範圍蔓延 (Scope Creep)**：專案在不受控制且未考慮到對預算及時程衝擊下，被**隨意增添交付功能與範疇**。

| 英文專有名詞 | 中文翻譯 | 考試核心記憶點 |
| :--- | :--- | :--- |
| **Strategic Management** | 策略管理 | 整合內外部環境與資源，制定目標以建立競爭優勢。 |
| **Value Chain Model** | 價值鏈模型 | 主要活動（進銷存客）與支援活動（後勤採購人資）。 |
| **BPR (Business Process Re-engineering)** | 企業流程再造 | 根本性重新設計業務流程，追求績效的大幅度飛躍。 |
| **Gantt Chart** | 甘特圖 | 簡單專案時程管控工具，**無法描繪活動相依性**。 |
| **PERT** | 計畫評核技術 | 運用**樂觀、悲觀、最可能時間**估計任務工期。 |
| **CPM** | 要徑法 | 計算關鍵路徑以決定專案**最短完工時間**。 |
| **Triple Constraints** | 三重限制 | 專案管理的限制因子：**成本、時間、範疇**。 |
| **Scope Creep** | 範圍蔓延 | 未考慮時程與預算下，隨意增添專案功能/範疇。 |
| **Management Accounting** | 管理會計 | 提供資訊給**內部使用者**（控制預算、改善缺失）。 |
| **Financial Accounting** | 財務會計 | 提供財務報告給**外部使用者**（遵循法規、歷史資訊）。 |
| **Audit** | 稽核 / 審計 | 企業執行內部控制，稽核人員應**獨立向董事會報告**。 |
| **Job Enlargement** | 工作擴大化 | 橫向工作擴張，增加相同難度的任務數量。 |
| **Job Enrichment** | 工作豐富化 | 縱向工作擴張，增加規劃與控制的職責與權限。 |
| **Job Rotation** | 工作輪調 | 員工在不同職位間輪換以學習新技能。 |
| **Social Engineering** | 社交工程 | 利用人性弱點（如欺騙、假冒）獲取機密資訊（屬資安）。 |
